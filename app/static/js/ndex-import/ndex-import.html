NDEx to Deckhard Importer

Instructions

1. Enter NDEx UUID

Provide the UUID of the NDEx network you want to import

2. Preview Network

Check network details and verify it's within size limits (max 100 nodes/edges)

3. Set Import Options

Choose whether to import nodes or edges and specify object type

4. Import

Creates an object_list with objects in Deckhard

Size Limits: Maximum 100 nodes or edges can be imported
